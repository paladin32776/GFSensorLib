# GFSensorLib

Sensor library for RaspberryPi.

Contains support for 
 - Bosch BME280 temprature, pressure, and humidity sensor.
 - BH1750 light sensor.
 - ADS1015 analog-digital converter

Usage:

See included 
  bme280_example.py 
and 
  ads1015_example.py.

Place GFSensorLib.py in the same folder.
